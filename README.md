# osi-licenses 

id-name pairs of OSI-approved licenses, based on list at http://opensource.org/licenses/alphabetical .

## Usage

```javascript
var licenses = require("osi-licenses")
if(licenses["MIT"] == "MIT license") {
  console.log("It worked!")
}
```
